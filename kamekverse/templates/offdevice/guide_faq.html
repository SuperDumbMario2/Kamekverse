Frequently Asked Questions (FAQ)

If you have any questions while using {{name}}, this is the first place to check for an answer.

I saw a post or comment that violates the {{name}} Code of Conduct. What should I do?

If you encounter a post or comment that violates the {{name}} Code of Conduct, please report the violation using the {{name}} reporting function.

Reported violations will be reviewed by a Nintendo moderator. Any post that violates the {{name}} Code of Conduct will be removed.

Please note: false reporting is a violation of the {{name}} Code of Conduct and is subject to action up to and including loss of {{name}} access.

I saw a comment with a spoiler in it, even though it wasn’t marked as having spoilers.

It’s up to users to mark their posts as having spoilers. However, if someone forgets to do this, you can help by reporting spoilers.

Comments and posts that receive enough spoiler reports will be marked as spoilers, viewable only to people who have chosen to keep spoilers visible.

Why do some users have a blue check mark? What are verified users? Can I be a verified user?

A verified user is someone whose identity has been authenticated and certified by the {{name}} team. Their verified status indicates that they are a Kamekverse moderator.

Verified-user status is granted only to parties when necessary as determined by {{name}} team. We usually do not accept requests for verification from the general public.

How do I delete a {{name}} post?

If you post something you later do not want other users to see, you can delete it. Here is how:

  1. Select the post you wish to delete from within {{name}}. If you need assistance locating the post:

    From within {{name}}, select your user icon, and then select Posts on the User Page.

  2. Scroll through the list of posts.
  3. Click on the post to open it's page.
  4. Select the wrench icon located in the lower-right corner of the post you want to delete.
  5. Select the drop-down menu, and select Delete.
  6. Select Submit.

Please note that deleting a post does not ensure complete or comprehensive removal of the content or information posted by you (for example, if another user has copied your post pr it was scraped or if you or another user previously shared the post on another website). We keep deleted posts for various purposes including preservation.
